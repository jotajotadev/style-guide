Colores en CSS

Para poder usar la nueva paleta de colores solo es necesario copiar el código que se muestra en la parte inferior. (Si en caso el proyecto se trabajará con un preprocesador de CSS o un Framework, es necesario adaptarlo y mantener los mismos códigos hexadecimales junto con el name-convention)

                        
                            :root {
                                /* Colors */
                                
                                /* --Primary */
                                --primary-400: #135B31;
                                --primary-300: #1C884A;
                                --primary-200: #25B562;
                                --primary-100: #A4ECC3;
                                --primary-50: #E1F9EB;
                                
                                /* --Neutral */
                                --neutral-400: #1E293B;
                                --neutral-300: #5A687D;
                                --neutral-200: #CBD5E1;
                                --neutral-100: #F8FAFC;
                                --neutral-50: #FFFFFF;
                                
                                /* --Amber */
                                --amber-400: #907349;
                                --amber-300: #D7AB6D;
                                --amber-200: #FDC980;
                                --amber-100: #FEDDAF;
                                --amber-50: #FFFBEB;
    
                                /* --Purple */
                                --purple-400: #411782;
                                --purple-300: #6D28D9;
                                --purple-200: #9333EA;
                                --purple-100: #E9D5FF;
                                --purple-50: #F5F3FF;
    
                                /* --Orange */
                                --orange-400: #C85C04;
                                --orange-300: #FA7305;
                                --orange-200: #FB923C;
                                --orange-100: #FDC79B;
                                --orange-50: #FEF1E6;
    
                                /* --Blue */
                                --blue-400: #1B5274;
                                --blue-300: #246893;
                                --blue-200: #2E8CC6;
                                --blue-100: #C6E1F2;
                                --blue-50: #E3F0F8;
    
                                /* --Yellow */
                                --yellow-400: #A16207;
                                --yellow-300: #CA8A04;
                                --yellow-200: #EAB308;
                                --yellow-100: #FEF08A;
                                --yellow-50: #FEFCE8;
    
                                /* --Red */
                                --red-400: #AD341F;
                                --red-300: #DC5138;
                                --red-200: #FF6347;
                                --red-100: #FFA999;
                                --red-50: #FFE9E5;
                            }
