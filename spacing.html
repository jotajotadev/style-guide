Sistema de espaciado

La presente guía muestra los espacios basados en 8 point grid, esto no significa que siempre tenga que ser escalable de 8px a 8px, al contrario es posible utilizar 4px en caso esa necesario.

Espacio en CSS

Para poder usar los espacios, es necesario usar margin, padding, width, height. Debido a que los espacios representan valores en px ejm:

                        
                            .selector {
                                width: var(--spacing-4);
                                /* es igual a decir: width: 16px;*/

                                height: var(--spacing-8);
                                /* es igual a decir: height: 48px;*/ 

                                margin-right: var(--spacing-11);
                                /* es igual a decir: margin-right: 96px;*/ 

                                padding-bottom: var(--spacing-2);
                                /* es igual a decir: padding-bottom: 8px;*/  
                            }
                        
                    
                        
                            :root {                            
                                /* Spacing */
                            
                                --spacing-1: 4px;
                                --spacing-2: 8px;
                                --spacing-3: 12px;
                                --spacing-4: 16px;
                                --spacing-5: 24px;
                                --spacing-6: 32px;
                                --spacing-7: 40px;
                                --spacing-8: 48px;
                                --spacing-9: 64px;
                                --spacing-10: 80px;
                                --spacing-11: 96px;
                                --spacing-12: 160px;
                            }
