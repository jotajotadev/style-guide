Sombras

La presente guía las sombras junto con el name convention. su principal uso es para dar elevación a ciertos elementos como: Cards. Articles. Sections, Dividers entre otros. En Wally el color “neutral-50 -> #FFFFFF” también se puede usar para dar elevación.

shadow-small

x: 0

y: 1

blur: 2

spread: 0

color: rgb(36, 104, 147, 0.05)

Sombras en CSS

Para hacer uso de las sombras solo es necesario copiar el código con el botón copiar que se muestra en la parte superior derecha.

                        
                            .shadow-small {
                                box-shadow: 0 1px 2px rgba(36, 104, 147, 0.05);
                            }
                            
                            .shadow {
                                box-shadow: 0 1px 3px rgba(36, 104, 147, 0.1);
                            }
                            
                            .shadow-middle {
                                box-shadow: 0 4px 6px -1px rgba(36, 104, 147, 0.1);
                            }
                            
                            .shadow-large {
                                box-shadow: 0 10px 15px -3px rgba(36, 104, 147, 0.1);
                            }
